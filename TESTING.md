# Beta Testing Notes

Please report problems by opening a GitHub issue.

Include:

- Razzia URL used
- Whether the extension was ON or OFF
- What happened
- What you expected
- Chrome version from chrome://settings/help
- Any notes that might help

Useful things to test:

- localhost manager page
- LAN IP manager page
- local hostname manager page
- HTTP domain manager page
- HTTPS domain manager page
- ON/OFF toggle reliability
- OFF cancels pending clicks
- Extension does not click when OFF
- Extension does not click anything except the Razzia Next button
