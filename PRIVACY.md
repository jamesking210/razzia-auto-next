# Privacy

Razzia Auto Next does not collect, transmit, sell, or share user data.

The extension stores only one setting locally in Chrome:

```text
ON/OFF state
```

That setting is stored with Chrome extension storage so the toolbar toggle remembers whether Auto Next is enabled or paused.

There is no server, no account system, no analytics, and no tracking.

The extension only runs on Razzia manager-style routes matched by the extension manifest:

```text
/manager
/party/manager
```
